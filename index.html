Mobile First Responsive Site

This is a responsive, mobile-first layout using HTML and CSS. The layout adjusts to accommodate small, medium, and large screen sizes. The page is styled with CSS for a small mobile device first. Then, using min-width media queries, breakpoints were added to adjust the layout for wider tablet and desktop screens.

Registration Form

SASS Style Guide

By converting the CSS into Sass the code was able to be split into several Sass partial files. Sass variables were created to simplify and reduce repeated styled throughout the site. Also, mixins were created to be used to create media queries for each breakpoint from small to large screen sizes.

Interactive Photo Gallery

The interactive photo gallery was built using JavaScript and jQuery. At the top of the page, a live search area is available where, upon typing, photos will hide and show depending on user input. When the user clicks on a thumbnail, the photo will display in a lightbox. There are also back and previous arrows to cycle through photos.

Visit Site Github

Wheel of Success

“Wheel of Success” is a word guessing game where players to try to guess a random phrase. Using Javascript, DOM Traversal and Event Bubbling are put on display. A player has five 'lives', and it the phrase is completed before they run out of lives, they win! 5 incorrect guesses, and the player loses. Win or lose, the player can start a new game as soon as they finish.

Visit Site Github

Web App Dashboard

This dashboard acts like your control panel for controlling your web app. Complete with JavaScript-driven charts, graphs, auto-complete search, local storage capabilities, and a traffic chart widget this dashboard has it all! Laid out with CSS Grid from top to bottom the dashboard is compatible with all screen sizes.

Visit Site Github

Employee Directory

Communicating with APIs allows this site to grab information for 12 random employees, and use that data to build a prototype for an employee directory. Initializing JSON object requests from the API using fetch and parsing the data allows the 12 employees that are listed to be displayed with their thumbnail image, full name, email, and location. Clicking the employee’s image or name will open a modal window with more detailed information.
